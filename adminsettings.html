Instalación

Instalación mediante archivo ZIP

1. Accede a tu sitio Moodle como administrador y ves a Administración del sitio > Plugins > Instalar plugins.
2. Carga el archivo ZIP con el código del plugin. Sólo se te pedirá que añadas detalles adicionales si el plugin no se detecta automáticamente.
3. Puedes descargar el ZIP del plugin desde https://github.com/UNIMOODLE/moodle-mod_hybridteaching
4. Comprueba el informe de validación del plugin y finaliza la instalación.

Instalación manual

Puedes instalar el plugin colocando el contenido en este directorio {your/moodle/dirroot}/mod/hybridteaching
Luego, inicia sesión en tu sitio Moodle como administrador y ves a Administración del sitio > Notificaciones para completar la instalación.

Alternativamente, puedes ejecutar $ php admin/cli/upgrade.php para completar la instalación desde la línea de comandos.
